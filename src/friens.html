Les freins

Le système de freinage permet de ralentir et d'arrêter le véhicule en toute sécurité.Il doit être :

  • Puissant
  • Stable
  • Progressif
  • Fiable dans toutes les conditions météorologiques (pluie, chaleur, froid, ...)

Comment ça fonctionne ?

  1. Quand tu appuies sur la pédale de frein, tu actionnes le maître-cylindre
  2. Celui-ci pousse du liquide de frein dans des tuyaux
  3. Le liquide transmet la pression instantanément (il est incompressible)
  4. Cette pression arrive aux étriers (ou cylindres de roue pour les tambours)
  5. Les plaquettes (ou mâchoires) sont pressées contre le disque (ou tambour)
  6. Le frottement ralentit la voiture

Les éléments principaux du système de freinage :

  1. Les disques de frein :
    • Plaques métalliques fixées aux roues
    • Les plaquettes viennent frotter dessus pour créer le freinage

    Quand les changer ?

    • Tous les 2 changements de plaquettes théoriquement
  2. Les plaquettes de frein :
    • Éléments en matériau composite qui pincent le disque
    • Elles s’usent avec le temps

    Quand les changer ?

    • Tous les 30 000 à 50 000 km selon l’usage
    • Quand la garniture est < 3 mm
    • Quand un bruit métallique apparaît
    • Quand le freinage devient moins efficace
    • Quand un voyant d’usure s’allume (sur certains modèles)
  3. Les étriers :
    • Ils contiennent les pistons qui poussent les plaquettes contre le disque
  4. Le liquide de frein :
    • Transmet la pression du pied jusqu’aux étriers
    • Sans lui → aucun freinage

    Quand le vidanger ?

    • Tous les 2 ans
  5. Le maître-cylindre :
    • Pompe hydraulique actionnée par la pédale
  6. L’ABS (système antiblocage) :
    • Empêche les roues de se bloquer lors d’un freinage d’urgence
    • Permet de garder le contrôle de la direction
  7. Les freins à tambour :

    8. Comment ça fonctionne ? À l’intérieur on trouve :

    • Deux mâchoires de frein
    • Des garnitures (matériau de friction)
    • Des ressorts de rappel
    • Un cylindre de roue (avec deux pistons)

    Avantages :

    • Moins chers → Fabrication simple
    • Très efficaces pour le frein à main
    • Bonne protection contre la saleté → tout est enfermé
    • Durée de vie longue

    Inconvénients :

    • Chauffent plus vite
    • Risque de fading
    • Moins performantes
    • Entretien plus complexe

Le frein à main (frein de stationnement) :

Il sert à immobiliser la voiture à l’arrêt, même sur une pente.Il empêche la voiture :

  • d’avancer
  • de reculer
  • de glisser en pente

Il y a 2 types de frein à main :

  1. Le frein à main mécanique (classique) :

    2. Actionné par un levier entre les sièges avant.Fonctionnement :

    • Tirez le levier
    • Un câble se tend
    • Il actionne les mâchoires dans les tambours ou les étriers (système intégré)
    • Les roues arrière se bloquent
  2. Le frein à main électrique :

    3. Le bouton “P” ou “AUTO HOLD” dans les voitures modernesFonctionnement :

    • Appuyez sur le bouton
    • Un moteur électrique serre les freins arrière
    • Le système se gère automatiquement (le calculateur vérifie la force)
    • Il peut s’activer tout seul à l’arrêt

Sur quoi agit le frein à main ?

  1. Sur les tambours :

    2. → Les mâchoires se bloquent contre le tambour

  2. Sur les disques :

    3. → Petit mécanisme intégré dans l’étrier

  3. Système “drum-in-disc” :

    4. → Petit tambour interne dans le disque
